Example User Stories

1. As a customer, I want to filter food items by allergens so that I can order safely.

Acceptance Criteria
• User can select allergen filters
• Menu updates based on selected filters
• User preferences can be saved

2. As a customer, I want to view order history so I can quickly reorder past meals.

Acceptance Criteria
• Order history is visible in profile
• User can reorder items from previous orders
