Agile Workflow Practice

This repository contains examples of Agile Practices including:

-User stories

-Acceptance criteria

-Sprint workflow concepts practiced using Jira

-Task tracking

-Backlog prioritization

Example user story:

As a customer, I want to filter food items by allergens so that I can order safely.

Acceptance Criteria:

-User can select allergens filters

-Menu updates based on selected filters

-System remembers user preferences
